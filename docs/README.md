## Quick Start

1. Buka aplikasi di browser → Login
2. Pergi ke **Bot Settings**
3. Isi **Wallet Address** + **Private Key** (wallet Arbitrum)
4. Enable **DeFi Trading** + **DeFi Only Scan**
5. Klik **Save All Settings**
6. Pergi ke **AI Analysis** → klik **Scan Manual**
7. Tunggu notifikasi Telegram `🔗 DeFi Trade Executed`
